2: Data Binding

The ng-model directive provides a method of data-binding. A data-binding in Angular is a type of dependency-injection. Underneath the hood, the ng-model uses a specialized type of Controller scoped to the ng-modelelement to create a new data object on the $scope object. Recall the $scope provides a means of communication between the Model and the View.
