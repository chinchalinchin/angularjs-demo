AngularJS is a framework for web development created and supported by Google. By using the Model-View-Control Architecture to organize itself, AngularJS modularizes and isolates the iteractings components of web applications. AngularJS prescribes a certain design; we must think in terms of Models, Views and Controllers. In AngularJS, a HTML template provides a View of a data Model. This interaction occurs through a Controller.

In other words, Views plug into Controllers so they can access and present the Models to the user.
