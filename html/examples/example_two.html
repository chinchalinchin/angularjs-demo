2: Data Binding

The ng-model directive provides a method of data-binding. A data-binding in Angular is a type of dependency-injection. Underneath the hood, the ng-model uses a specialized type of Controller bound to the ng-modelelement to create a new data object on the $scope object. Recall the $scope provides a means of communication between the Model and the View.

Example 1

Simple Data Binding allows you to directly manipulate the Document-Object-Model. In this example, you can dynamically alter the table element in the column next to the input box by typing into it. Your input will change the DOM.
